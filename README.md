## Run Locally

Clone the project

```bash
  git clone https://link-to-project](https://github.com/your-username/smspam.git
```

Go to the project directory

```bash
  cd smspam
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Run on browser

```bash
  python app.py
```
